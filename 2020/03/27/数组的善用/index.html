一

输入两个非负 10 进制整数 A 和 B (≤2
​30
​​ −1)，输出 A+B 的 D (1<D≤10)进制数。

输入格式：

输入在一行中依次给出 3 个整数 A、B 和 D。

输出格式：

输出 A+B 的 D 进制数。

输入样例：

123 456 8

输出样例：

1103

思路： 1.十进制如何转化为其他进制？

对基取余法

123+456=579

579%8=72 ····3

72%8=9 ·····0

9%8=1 ·····1

1%8=0 ·····1

2.如何才能输出余数，采用数组存储，进行从低位到高位的枚举，最后再从高位到低位进行输出

`#include <stdio.h>
 int main()
{
int A, B, D,sum=0;
scanf("%d%d%d", &A, &B, &D);
sum = A + B;
int a[31], num = 0;
do
{
    a[num++] = sum%D;
    sum = sum / D;
} while (sum != 0);
for (int i = num-1; i >= 0; i--)
{
    printf("%d", a[i]);
}
return 0;
}`

二

让我们用字母 B 来表示“百”、字母 S 表示“十”，用 12…n 来表示不为零的个位数字 n（<10），换个格式来输出任一个不超过 3 位的正整数。例如 234 应该被输出为 BBSSS1234，因为它有 2 个“百”、3 个“十”、以及个位的 4。

输入格式：

每个测试输入包含 1 个测试用例，给出正整数 n（<1000）。

输出格式：

每个测试用例的输出占一行，用规定的格式输出 n。

输入样例 1：

234

输出样例 1：

BBSSS1234

输入样例 2：

23

输出样例 2：

SS123

思路：对测试例求余用数组保存，从高位到低位进行枚举输出

`#include <stdio.h>
 int main()
{
int n;
int a[5], num = 0;
scanf("%d", &n);
while (n != 0)
{
    a[num] = n % 10;
    num++;
    n = n / 10;
}
for (int i=num-1;i>=0;i--)
{
    if (i == 2)
    {
        for (int j = 0; j < a[i]; j++)
            printf("B");
    }
    else if (i == 1) {
        for (int j = 0; j < a[i]; j++)
            printf("S");
    }
    else
        for (int j = 1; j <=a[i]; j++)
        {
            printf("%d",j);
        }
}
return 0;
}`

输入格式：

每个输入包含 1 个测试用例，即一个不超过 1000 位的正整数 N。

输出格式：

对 N 中每一种不同的个位数字，以 D:M 的格式在一行中输出该位数字 D 及其在 N 中出现的次数 M。要求按 D 的升序输出。

输入样例：

100311

输出样例：

0:2

1:3

3:1

思路：

1.以字符数组形式存储数据，记录长度

2采用字符相减，字符相减等同于ASCII的相减，由于输入例都是int， a[0]-‘0’=1

3采用枚举从低到高输出

`#include <stdio.h>
 #include <string.h>
 int main()
{
char c[1010];
scanf("%s",c);
int len = strlen(c);
int a[10] = { 0 };
for (int i = 0; i < len; i++)
{
    a[c[i] - '0']++;
}
for (int i = 0; i < 10; i++)
{
    if (a[i] != 0)
    {
        printf("%d:%d\n", i, a[i]);
    }
}
return 0;
}`
