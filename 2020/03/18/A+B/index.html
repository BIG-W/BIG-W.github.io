A + B
发表于 2020-03-18| 更新于 2020-03-22|算法A + B
字数总计:857|阅读时长: 3 分钟
| 阅读量:

在基础算法中,A+B类的题型很常见，从最基础普通到各类变种，考察的知识点也各不相同，下面将依次举例

  1. 基础型（A+B和C）

给定区间 [−2
​31
​​ ,2
​31
​​ ] 内的 3 个整数 A、B 和 C，请判断 A+B 是否大于 C。
输入格式：

输入第 1 行给出正整数 T (≤10)，是测试用例的个数。随后给出 T 组测试用例，每组占一行，顺序给出 A、B 和 C。整数间以空格分隔。

输出格式：

对每组测试用例，在一行中输出 Case #X: true 如果 A+B>C，否则输出 Case #X: false，其中 X 是测试用例的编号（从 1 开始）。

输入样例：

4

1 2 3

2 3 4

2147483647 0 2147483646

0 -2147483648 -2147483647

输出样例：

Case #1: false

Case #2: true

Case #3: true

Case #4: false

对于此类题型，无需思考，直接硬解，唯一考虑的是变量的取值范围，int 取值范围 -2的31次方~2的31次方-1 也就是大概在-2×10的9次方~2×10的9次方
long long 取值范围 -2的63次方~2的63次方-1 大概在-9×10的18次方~9×10的18次方
代码如下：

`#include <stdio.h>
 long long A, B, C;
 int main()
{
int T, tep=1;
scanf("%d",&T);
for(int i=0;i<T;i++)
{
    scanf("%lld%lld%lld",&A,&B,&C);
    if (A + B > C)
    {
        printf("Case #%d: true\n", tep++);
    }
    else
    {
        printf("Case #%d: false\n", tep++);
    }
}
return 0;
 }`

2.变种型（A + B 和 C （64bit））

Given three integers A, B and C in [−2
​63
​​ ,2
​63
​​ ], you are supposed to tell whether A+B>C.

Input Specification:

The first line of the input gives the positive number of test cases, T (≤10). Then T test cases follow, each consists of a single line containing three integers A, B and C, separated by single spaces.

Output Specification:

For each test case, output in one line Case #X: true if A+B>C, or Case #X: false otherwise, where X is the case number (starting from 1).

Sample Input:

3

1 2 3

2 3 4

9223372036854775807 -9223372036854775808 0

Sample Output:

Case #1: false

Case #2: true

Case #3: false

此题则需要考虑两数之和是否溢出问题，此时重点便是两个正数之和等于负数或是两个负数之和等于正数，便是溢出，这一点也是笔者学习后知道的… 这一点在计算机组成原理中有说明， 怪我自己没学好。
①.A+B>=2的63次方，结果A+B>C肯定成立，但相加后的结果肯定会正溢出，而正溢出值得区间为[-2的63次方，-2]
②.A+B<-2的63次方，A+B<C肯定成立，同理会负溢出，负溢出区间[0,2的63次方]
全部结果的输出，则用bool代替，true,false.

代码如下：

`#include <stdio.h>
 #define true 1
 #define false 0
int main()
{
int n;
bool flag;
int tep = 1;
scanf("%d", &n);
for (int i=0;i<n;i++)
{
    long long a, b, c,res;
    scanf("%ld%ld%ld",&a,&b,&c);
    res = a + b;
    if (a > 0 && b > 0 && res < 0) { flag = 1; }
    else if (a < 0 && b < 0 && res>=0) { flag = 0; }
    else if (a + b > c) { flag = 1; }
    else  flag = 0; 
    if (flag == 1) {
        printf("Case #%d: true", tep++);
    }
    else
        printf("Case #%d: false",tep++);
}
return 0;
}`
